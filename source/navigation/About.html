ABOUT DEUS-FALA

Deus-Fala is a Documentation based on HTML and CSS which compiles Catholic Prayers, Scriptures and Various Biblical Content. This documentation is 100% Free and Open Source. The documentation is easy to navigate and is carefully organized into different categories. The Source Code underlying this documentation is licensed under the MIT License.
